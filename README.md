### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AbdallahRaslan/SPAquotes.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Run Start 
   ```sh
   npm start
   ```

4. Replace the link in the fetch method with your firebase runtime link, you will find it in one file :
   ```sh
   lib/api.js
   ```
